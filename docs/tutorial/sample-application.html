# サンプルアプリについて

今回のチュートリアルで作成するアプリケーションを紹介します。

リアルタイム検索アプリケーションです。

左（Search）の入力欄に打ち込んだ文字を含む名前のみ、右（Members）に表示されます。

文字を入力するたびに Ajax で検索を行い、結果を HTML に反映しています。

Member を追加することもできます。
